PHPMailer Library - DOWNLOAD REQUIRED
======================================

Download the following 3 files from GitHub and place them in this folder:

1. PHPMailer.php
2. SMTP.php
3. Exception.php

Download from: https://github.com/PHPMailer/PHPMailer/tree/master/src

Direct links:
- https://raw.githubusercontent.com/PHPMailer/PHPMailer/master/src/PHPMailer.php
- https://raw.githubusercontent.com/PHPMailer/PHPMailer/master/src/SMTP.php
- https://raw.githubusercontent.com/PHPMailer/PHPMailer/master/src/Exception.php

After downloading, your folder structure should be:
/public/phpmailer/
  - PHPMailer.php
  - SMTP.php
  - Exception.php
  - README.txt (this file)
